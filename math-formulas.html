Key Statistical Formulas

Standard Deviation

\[ \sigma = \sqrt{\frac{\sum (x_i - \mu)^2}{N}} \]

Z-Score

\[ Z = \frac{X - \mu}{\sigma} \]

Multiple Regression

\[ Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \dots + \beta_n X_n + \epsilon \]

Present Value

\[ PV = \frac{FV}{(1 + r)^n} \]
